## ğŸš€ Getting Started

### Prerequisites

Make sure you have the following installed on your system:

- **Node.js** (v18 or higher) - [Download](https://nodejs.org/)
- **npm** (comes with Node.js) or **bun** - [Download Bun](https://bun.sh/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Riyakoushik/MY-PERSIONAL-WEBSITE--DARK-THEAM-.git
   cd MY-PERSIONAL-WEBSITE--DARK-THEAM-
   ```

2. **Install dependencies**
   
   Using npm:
   ```bash
   npm install
   ```
   
   Or using Bun (faster):
   ```bash
   bun install
   ```

3. **Start the development server**
   
   Using npm:
   ```bash
   npm run dev
   ```
   
   Or using Bun:
   ```bash
   bun run dev
   ```

4. **Open your browser**
   
   Navigate to `http://localhost:5173` to see the website running locally.

## ğŸŒ Deployment

This portfolio is configured for deployment on multiple platforms:

The project includes `netlify.toml` for Netlify deployment. Simply connect your GitHub repository to Netlify for automatic deployments.
